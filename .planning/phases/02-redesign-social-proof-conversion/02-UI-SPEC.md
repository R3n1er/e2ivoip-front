## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, badge padding |
| md | 16px | Default element spacing, card internal padding |
| lg | 24px | Section internal padding, card content gaps |
| xl | 32px | Layout gaps between components |
| 2xl | 48px | Section vertical padding (py-12) |
| 3xl | 64px | Major section breaks (py-16) |
| 4xl | 80px | Hero/CTA section vertical padding (py-20) |
| 5xl | 96px | Page-level section padding (py-24) |

Exceptions:
- Touch targets for phone links: minimum 44px height (accessibility)
- `.monolith-btn` inner span: `px-10 py-4` (40px x 16px) for primary, `px-12 py-5` (48px x 20px) for large variant

## Typography

4 sizes, 2 weights only.

| Role | Size | Weight | Line Height | Tracking | Transform |
|------|------|--------|-------------|----------|-----------|
| Body | 16px (text-base) | 400 (font-normal) | 1.625 (leading-relaxed) | normal | none |
| Body secondary / Nav link | 14px (text-sm) | 400 for body-secondary, 900 for nav | 1.5 | body-secondary: normal; nav: -0.03em | nav: uppercase |
| CTA label / Micro-label | 12px (text-xs) | 900 (font-black) | 1.5 | CTA: 0.2em; Micro: 0.3em | uppercase |
| Heading (section / display) | 36px (text-4xl) / 48px (text-5xl) | 900 (font-black) | 0.95 (leading-[0.95]) | section: -0.04em; display: -0.03em | display: uppercase |

**Size count:** 4 (12px, 14px, 16px, 36/48px responsive heading pair)

**Weight count:** 2 (400 normal, 900 black)

### C-03: CTAButton Refactor (Link-only)

**Source decision:** D-07, D-12, CONV-01, CONV-02

**Current violations to fix:**
1. `<button>` nested inside `<Link>` -- FORBIDDEN by CLAUDE.md
2. `shadow-md hover:shadow-xl` -- FORBIDDEN by Monolithe rules
3. `hover:-translate-y-0.5` -- not Monolithe (should be hard shadow shift)
4. Gradient overlay on hover -- FORBIDDEN by Monolithe rules
5. `font-semibold text-lg` -- should be `font-black uppercase tracking-[0.2em] text-xs` (CTA label style)

**Refactored pattern (matching Hero/Contact section pattern):**

```tsx
// Primary CTA
<Link
  href={href}
  className="monolith-btn"
  onClick={handleTrackEvent}
>
  <span className="block bg-red-primary text-white px-10 py-4 text-sm font-black uppercase tracking-[0.2em]">
    {icon && <i className={`lni ${icon} mr-3`} />}
    {children}
  </span>
</Link>

// Secondary CTA (white bg)
<Link
  href={href}
  className="monolith-btn"
  onClick={handleTrackEvent}
>
  <span className="block bg-white text-[#091421] px-10 py-4 text-sm font-black uppercase tracking-[0.2em]">
    {icon && <i className={`lni ${icon} mr-3`} />}
    {children}
  </span>
</Link>
```

**For tel: links (external):**

```tsx
<a
  href="tel:+590590123456"
  className="monolith-btn"
  onClick={handlePhoneTrackEvent}
>
  <span className="block bg-red-primary text-white px-10 py-4 text-sm font-black uppercase tracking-[0.2em]">
    <i className="lni lni-phone mr-3" />
    0590 12 34 56
  </span>
</a>
```

| Property | Value |
|----------|-------|
| Outer element | `<Link>` for internal, `<a>` for tel:/mailto: |
| Class on outer | `monolith-btn` only |
| Inner element | `<span className="block">` |
| Inner bg primary | `bg-red-primary text-white` |
| Inner bg secondary | `bg-white text-[#091421]` |
| Typography | `text-sm font-black uppercase tracking-[0.2em]` |
| Padding | `px-10 py-4` (standard) or `px-12 py-5` (large) |
| Hover | Handled by `.monolith-btn` CSS: translate(2px,2px) + shadow shift |
| Active | Handled by `.monolith-btn` CSS: translate(5px,5px) + shadow collapse |
| Arrow icon | REMOVED -- not part of Monolithe CTA pattern (hero/contact don't use it) |
| Gradient overlay | REMOVED -- violation |

## Accessibility

| Concern | Contract |
|---------|----------|
| Phone links | `<a href="tel:...">` with visible text showing number |
| CTA buttons | Sufficient color contrast: white on #E53E3E = 4.63:1 (passes AA large text) |
| Star ratings | Decorative only -- no alt text needed (visual reinforcement of testimonial) |
| Section headings | Proper `<h2>` / `<h3>` hierarchy within sections |
| Motion | `.monolith-btn` hover uses `transition: transform 0.1s` -- brief, not problematic for reduced-motion |
| Touch targets | All interactive elements minimum 44px in smallest dimension |
